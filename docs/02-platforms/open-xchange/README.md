# Open-Xchange Research

## Status

**Feasibility and architecture research only.**

I did not deploy or claim a working Open-Xchange platform.

## Main finding

OX App Suite provides groupware/webmail functionality, but it is not automatically a complete multi-domain mail-hosting administration platform. A self-hosted solution also needs SMTP, IMAP, database, filtering, provisioning, security, backup and monitoring.
