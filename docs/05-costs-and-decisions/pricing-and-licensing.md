# Pricing and Licensing

## Mailcow

Mailcow core is GPLv3 and does not require a separate per-user “Pro edition” licence. Commercial support or managed operation is a separate service.

## Cost categories

- compute;
- persistent storage;
- backups/snapshots;
- public IP and egress;
- domain/DNS;
- monitoring;
- administration;
- migration;
- security response;
- recovery testing;
- commercial support.

## Pricing rules

- state mailbox and storage limits;
- define retention;
- define included support hours;
- define response expectations;
- price migrations separately;
- avoid unlimited storage/support;
- review after measured usage;
- separate infrastructure cost from engineering value.

Open-source software can reduce licence cost. It does not remove the need for skilled operation and recovery.
